## 🌟 About FarmConnect Kenya

**FarmConnect Kenya** is a revolutionary digital platform designed to bridge the gap between Kenya's 6 million smallholder farmers and profitable markets. Our mission is to transform the agricultural value chain by providing farmers with direct market access, real-time pricing data, essential agricultural services, and comprehensive financial solutions.

### 🎯 The Problem We Solve

- 📉 **40-60% below market value** - Current farmer earnings due to middlemen exploitation
- 📊 **Low productivity** - Only 1.2 tons per hectare vs. global average of 5.9 tons
- 🔍 **Information gaps** - Limited access to real-time market data
- 🤝 **Market access barriers** - Difficulty connecting with verified buyers
- 💰 **Financial exclusion** - Limited access to agricultural credit and insurance
- 👨‍🌾 **Youth migration** - Young people leaving agriculture for urban areas

## 🎯 M-Pesa Integration Technical Details

### 🔧 Payment Processing Flow
1. **STK Push Initiation**: User initiates payment through platform interface
2. **Safaricom API Call**: Secure API request to M-Pesa servers
3. **Mobile Prompt**: User receives payment prompt on their mobile device
4. **PIN Entry**: User enters M-Pesa PIN to authorize transaction
5. **Callback Processing**: Platform receives transaction confirmation
6. **Database Update**: Transaction status and receipt number stored
7. **User Notification**: Success confirmation with transaction details

### 🛠️ Technical Implementation
```typescript
// M-Pesa STK Push Implementation
const initiatePayment = async (phoneNumber, amount, description) => {
  const response = await supabase.functions.invoke('mpesa-stk-push', {
    body: {
      phone_number: phoneNumber,
      amount: amount,
      account_reference: 'FarmConnect',
      transaction_desc: description
    }
  });
  return response.data.CheckoutRequestID;
};
```

### 🔒 Security Features
- **OAuth Token Management**: Automatic token refresh and management
- **Callback URL Verification**: Secure webhook endpoint for transaction updates
- **Data Encryption**: All sensitive data encrypted in transit and at rest
- **Audit Logging**: Comprehensive transaction logging for compliance

## 🛠️ Getting Started

### Prerequisites 📋
- 📦 **Node.js** (v18 or higher)
- 🧶 **npm** or **yarn** package manager
- 💻 **Modern web browser** (Chrome, Firefox, Safari, Edge)
- 📱 **Mobile device** for M-Pesa testing (optional)

### Quick Installation 🔧

1. **Clone the repository** 📥
```bash
git clone https://github.com/your-username/farmconnect-kenya.git
cd farmconnect-kenya
```

2. **Install dependencies** 📦
```bash
npm install
# or
yarn install
```

3. **Start development server** 🚀
```bash
npm run dev
# or
yarn dev
```

4. **Open your browser** 🌐
Navigate to `http://localhost:5173`

### Building for Production 🏗️

```bash
npm run build
# or
yarn build
```

### Environment Setup 🔧

The platform uses Supabase for backend services. Key environment variables are managed through Supabase secrets:

- `MPESA_CONSUMER_KEY`: Safaricom M-Pesa API consumer key
- `MPESA_CONSUMER_SECRET`: Safaricom M-Pesa API consumer secret
- `MPESA_BUSINESS_SHORTCODE`: M-Pesa business shortcode for payments
- `MPESA_PASSKEY`: M-Pesa API passkey for transaction security
