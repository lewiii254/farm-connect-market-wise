## 🌟 About FarmConnect Kenya

**FarmConnect Kenya** is a revolutionary digital platform designed to bridge the gap between Kenya's 6 million smallholder farmers and profitable markets. Our mission is to transform the agricultural value chain by providing farmers with direct market access, real-time pricing data, and essential agricultural services.

### 🎯 The Problem We Solve

- 📉 **40-60% below market value** - Current farmer earnings due to middlemen exploitation
- 📊 **Low productivity** - Only 1.2 tons per hectare vs. global average of 5.9 tons
- 🔍 **Information gaps** - Limited access to real-time market data
- 🤝 **Market access barriers** - Difficulty connecting with verified buyers

## 🛠️ Getting Started

### Prerequisites 📋
- 📦 **Node.js** (v18 or higher)
- 🧶 **npm** or **yarn**
- 💻 Modern web browser

### Installation 🔧

1. **Clone the repository** 📥
```bash
git clone https://github.com/your-username/farmconnect-kenya.git
cd farmconnect-kenya
```

2. **Install dependencies** 📦
```bash
npm install
# or
yarn install
```

3. **Start development server** 🚀
```bash
npm run dev
# or
yarn dev
```

4. **Open your browser** 🌐
Navigate to `http://localhost:5173`

### Building for Production 🏗️

```bash
npm run build
# or
yarn build
```
